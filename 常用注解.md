# 全局异常处理器
## @RestControllerAdvice
- 加在类上
- @ConrollerAdvice + @ResponseBody 表明全局异常处理类 + 返回给前端json格式数据
## @ExceptionHandler
- 加在方法上
- 方法形参表明要处理的异常类型

``` java
@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler{
  @ExceptionHandler
  public Result handleException(Exception e){
    log.error("日志记录",e)
    return Result.error("result格式的错误提示信息")
  }
}
```
